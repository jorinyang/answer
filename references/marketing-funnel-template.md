# 运营营销方案 — 漏斗数学模板

> 用于 answer 新方案领域的营销/运营类方案。每次制定 GMV 导向的营销计划时，按此模板做漏斗反推。

## 漏斗反推公式

```
目标 GMV = 付费客户数 × 客单价
              ↑ 私域→支付转化率
需私域池     = 付费客户数 ÷ 转化率
              ↑ 渠道→私域通过率（1 - 流失率）
需渠道触发   = 私域池 ÷ 通过率
```

**常用假设（户外体验类，待行业验证）**：
- 抖音→企微流失率：30%（贵州之客已验证）
- 企微→支付转化率：50-70%（精准加微用户）
- 大V分润模式：按有效转化订单分润，零预付坑位费

## 渠道矩阵模板

| 渠道 | 角色 | 内容量 | 加微占比 | 投放 | 内容类型 |
|------|------|--------|---------|------|---------|
| 抖音 | 主攻 | 18-20条/月 | 60-70% | DOU+ 50-60%预算 | 场景型/人设型/转化型 |
| 视频号 | B端入口 | 4-6条/月 | 10-15% | — | 研学案例/基地展示 |
| 小红书 | 种草长尾 | 8-10篇/月 | 10-15% | 薯条 20%预算 | 攻略/体验/亲子/合集 |
| 公众号 | 品牌背书 | 4篇/月 | 5% | — | 品牌故事/客户案例/转化 |

## 内容复用流

```
拍摄 1次 → 剪辑师标记高光片段
  → 抖音：快节奏混剪（15-60s）
  → 视频号：同素材 + 配音解说（30-90s）
  → 小红书：精选照片 + 短文案
  → 公众号：嵌入文章配图
```

## 私域承接时间线

| 节点 | 动作 |
|------|------|
| 加微当天 | 自动欢迎语 + 确认兴趣分类（个人/家庭/企业） |
| Day 3 | 未回复者推送「限时优惠」提醒 |
| Day 7 | 未转化者推送「客户体验视频/好评截图」 |
| Day 14 | 最后触达 → 转入长期培育池 |

## 预算分配参考（户外体验类）

| 项目 | 占比 |
|------|------|
| 主渠道投放 | 50-60% |
| 次级渠道投放 | 20% |
| 物料/礼品 | 10% |
| 应急机动 | 10% |

> 大V合作建议走分润制（0 预付 + 按单分佣），降低现金风险。
