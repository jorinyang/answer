# 培训/工作坊方案模板

> 当 📋 新方案 领域下产出的是「培训课程方案」或「工作坊方案」时，使用此模板替代通用方案结构。

## 适用场景

- 企业AI培训方案（如赛前辅导、DEAP工作坊、Agent搭建培训）
- 客户要求单文档交付，含课程设计+报价+论证
- 需要从大量输入场景中筛选课程案例

## 方案结构（6章+附录）

```
1. 执行摘要                    ← 管理层入口：1页读完（产出+报价+为什么）
2. 场景诊断                    ← 从输入场景清单提炼洞察（热力图+类型分布+成熟度）
3. 课程方案
   ├── 3.1 课程目标与产出定义
   ├── 3.2 前期远程准备
   └── 3.3 Day 1 详细日程（表格式：时间|模块|内容|产出）
4. 为什么这样设计               ← 正面回应客户核心担忧（每担忧1节）
5. 预算与交付清单              ← 费用明细表+交付清单表
6. 关于我们                    ← 供应商简介（1-2段）
附录：场景-Agent映射表          ← 全量场景→Agent类型+难度+关键步骤
```

## 关键设计决策

### 课程案例筛选

1. 对输入场景做 RICE 评分（自动化+人工复核）
   - R（Reach）：基于推广范围（全公司5→本岗位1）
   - I（Impact）：基于效率提升百分比
   - C（Confidence）：基于已有AI尝试程度
   - E（Effort）：基于平台搭建复杂度（Fibonacci 1/2/3/5/8）
2. 从高分中选 6-8 个场景，兼顾：
   - 类型多样性（不全是同类型）
   - 难度梯度（上午E≤2，下午E=2-3）
   - DEAP技能覆盖（知识库/RAG/文档解析/数据分析/模板匹配）

### 三大担忧回应模式

每个担忧用三段式：
1. **正面回应**（一句话结论，加粗前置）
2. **证明逻辑**（2-3条论据，有数据/对比更佳）
3. **保障措施**（具体机制，不用模糊措辞）

### 场景-Agent映射表

至少覆盖全部输入场景，每行标注：
- 场景名称
- Agent类型（知识问答/文档审查/分析Agent/视觉Agent等）
- 难度（★=30分钟 ★★=1小时 ★★★=2小时+ ★★★★=需开发支持）
- 关键步骤（2-3个关键词，如"知识库+Agent配置"、"数据MCP+报告模板"）

### 预算结构

培训/工作坊类方案定价公式：
```
总预算 = 人天费用 + 差旅 + 物料 + 平台预配置
```
- 人天 = 讲师天数 × 费率 + 助教天数 × 费率
- 差旅 = 人数 × 天数 × 单日费率
- 物料 = 培训手册 + 演练环境
- 平台预配置含在前期准备人天内，不单独计费

## 参考案例

- **启源芯动力AI赛前辅导**（2026-06-01）：72场景→RICE Top 8，1天DEAP工作坊，¥84,500
- **浙能燃气人资AI工作坊**（2026-06-01）：DEAP工作坊 vs 自研开发双方案对比

## 常见陷阱

| 陷阱 | 避免方式 |
|------|---------|
| 课程内容写成"我们讲什么" | 写成"学员能做什么"，每行表格式日程必须有"产出"列 |
| **写成"讲师备课用的提词器/教学备注"** | **写成讲师面对学员要说的逐字稿——讲师拿起文档就能念。不要写"此处讲师引导学员思考"——直接写讲师要说的话。文档=讲师嘴里的内容，不是脑中的教案。** |
| 担忧回应回避问题 | 不写"我们经验丰富可以解决"，写具体机制 |
| 场景映射表不完整 | 全部输入场景必须出现在映射表中，不能只映射Top 8 |
| 报价含糊 | 精确到百元，附计算明细，每行可复算 |
