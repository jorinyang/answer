# 钉钉 DEAP 企业AI平台

> 钉钉 Digital Enterprise AI Platform — 类似阿里云百炼/扣子/Dify 的企业级智能体搭建平台。
> 来源：钉钉官方文档 CP 提取（https://alidocs.dingtalk.com/i/p/Y7kmbokZp3pgGLq2/docs/Y1OQX0akWm3g2GBdsALQqNlrJGlDd3mE）

## 平台定位

DEAP 是钉钉生态内的企业级 AI Agent 搭建平台，提供从知识库→Agent搭建→评测→发布→运营的全生命周期管理。

## 模块矩阵

| 模块 | 能力 | 对应需求 |
|------|------|---------|
| **智能体开发** | 可视化 Agent 搭建（提示词+RAG绑定+工具配置） | 所有 Agent 创建 |
| **智能体评测** | Agent 效果评估体系（评测集+自动测试） | 质量保障 |
| **企业技能中心** | 官方技能 / 三方技能 / 自定义 MCP 插件 / 自定义工作流 | 系统对接+业务逻辑 |
| **AI知识管理** | 知识集 + 评测集 + 训练集（完整 RAG 管道） | 16 个知识库模块 |
| **模型管理** | 模型广场（公有云模型）+ 企业模型（自有接入+专属训练） | 内网 Qwen/DeepSeek 接入 |
| **运营中心** | 使用量统计、命中率监控、满意率追踪 | 运维监控 |
| **安全与权限** | 用户授权、敏感词管控、PII 脱敏 | 安全合规 |
| **开放 API** | 程序化调用接口 | 二次开发 |

## 适用场景判断

✅ **推荐使用 DEAP：**
- Agent 数量 ≥ 10 个，手动开发不经济
- 需求包含大量 RAG 知识库（DEAP 内置管道）
- 需要多系统 MCP 对接（DEAP 技能中心）
- 客户希望长期自维护和扩展 Agent

❌ **不推荐 DEAP：**
- Agent 数量 ≤ 3 个（平台费不划算）
- 预算极低（平台费就吃掉大半）
- 客户完全无 IT 能力（无法学会低代码搭建）

## 与自研开发对比

| 维度 | DEAP 工作坊 | 自研开发 |
|------|-----------|---------|
| Agent 创建 | 可视化配置 | 写代码 |
| RAG 管道 | 内置 | 自建 |
| MCP 对接 | 技能中心注册 | 逐个开发 |
| 权限体系 | 内置 | 自研 |
| 多 Agent 协作 | 工作流编排 | 自研消息总线 |
| 学习曲线 | 2-3 天培训可上手 | 需专业开发团队 |

## 工作坊模式下的 Agent 搭建流程

```
知识库准备 → DEAP 上传文档 → 自动向量化 → 创建知识集
    ↓
Agent 配置 → 选模型 → 绑定知识库 → 写提示词 → 配工具/技能
    ↓
技能绑定 → 技能中心 → MCP 插件注册 → 测试连通
    ↓
评测验证 → 评测集测试 → RAG 命中率检查 → 上线
```
